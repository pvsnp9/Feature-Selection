
#### Constant Feature 

**Removing constant features:**
 
1. **variance threshold - sklearn**
    1. Store constant feature 
    2. Numerical variables 

2. **Std() - pandas**
    1. Quick 
    2. Does not store constant features 
    3. Numerical vars 

3. **nunique() - pandas**
    1. Quick 
    2. Does not store constant features 
    3. Numerical and categorical variables
